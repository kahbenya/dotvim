README
------

.vim folder and plugins managed by pathogen

INSTALL
-------
Clone the repo
> 
    git clone https://github.com/kahbenya/dotvim.git ~/.vim

Initialize and update submodules

> 
    cd .vim
    git submodule init
    git submodule update

Symlink vimrc file

> 
    cd ~
    ln -s .vim/vimrc .vimrc
